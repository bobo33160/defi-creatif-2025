# defi-creatif-2025
Site web gratuit générant un défi créatif quotidien sur un an. Mode sombre, formulaire sécurisé pour proposer des défis, historique local, export texte, notifications rappelleurs. Simple à déployer sur GitHub Pages avec lien vers page de dons. Facile, sécurisé et engageant.


# Défi Créatif Quotidien - Avancé

## Description

Ce site web statique gratuit propose un défi créatif par jour pendant environ un an afin de stimuler votre imagination et votre créativité. Il comprend :

- Un générateur de défis variés (écriture, dessin, programmation, DIY, photographie, etc.)
- Mode sombre activable pour un confort visuel optimal
- Formulaire sécurisé avec CAPTCHA pour proposer vos propres défis
- Historique local des défis réalisés, sauvegardé dans le navigateur (localStorage)
- Export de votre journal personnel de défis au format texte
- Rappels quotidiens par notifications navigateur (optionnels)
- Lien clair vers une page de dons pour soutenir le créateur

Le site est entièrement en HTML, CSS et JavaScript, sans backend, garantissant simplicité, sécurité et respect de la vie privée. Il est idéal pour être hébergé gratuitement, notamment via GitHub Pages.

---

## Installation et déploiement

1. Clonez ou téléchargez ce dépôt.
2. Remplacez l'URL de la page de dons dans le fichier `index.html` par votre propre lien.
3. Hébergez le site gratuitement, par exemple sur [GitHub Pages](https://pages.github.com/) (voir guide dans ce dépôt).
4. Ouvrez le site dans un navigateur moderne (Chrome, Firefox, Edge, Safari).

---

## Fonctionnalités

- Génération aléatoire d'un défi quotidien
- Formulaire pour ajouter vos propres défis (avec validation et CAPTCHA)
- Historique de tous vos défis réalisés stocké localement
- Export facile de l'historique au format `.txt`
- Notifications journalières personnalisables par l’utilisateur
- Mode sombre activable avec mémorisation de la préférence
- Accessibilité renforcée avec ARIA
- Lien vers la page de dons pour soutenir le projet

---

## Sécurité et confidentialité

- Pas de collecte ni stockage de données côté serveur
- CAPTCHA simple pour éviter les spams dans le formulaire
- Tous les contenus sont stockés localement dans le navigateur
- Application statique, ce qui limite considérablement les vecteurs d’attaque
